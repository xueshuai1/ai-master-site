## 💡 给开发的具体建议

### 本次执行（最高优先级 1-4）
1. `git pull origin main` 拉取最新代码
2. 更新 `src/data/update-time.ts` 中的首页时间戳为 `2026-04-14`
3. 修改首页新闻卡片，包裹 `<Link>` 组件，点击跳转到对应新闻页
4. 修改博客卡片，添加 `onClick` 或 `<Link>` 实现跳转
5. 修改 Footer 组件，将 `href="#"` 替换为实际链接
6. `npm run build` 验证编译
7. `git commit && git push` 推送
8. 运行 `./scripts/deploy.sh` 更新部署

### 代码改动原则
- 最小改动优先，不要重构
- 每次改动后立即 `npx tsc --noEmit` 验证
- 不要破坏现有功能
- 所有改动必须编译通过才能推送
