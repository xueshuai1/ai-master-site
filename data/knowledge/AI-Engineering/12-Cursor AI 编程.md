# Cursor AI 编程

## 核心概念

Cursor 是一款基于 AI 的代码编辑器，将大语言模型深度集成到开发工作流中。它支持自然语言代码编辑、智能补全、代码生成和调试等功能，是新一代 AI 原生开发工具的代表。

### Cursor 核心特性

```mermaid
graph TB
    A[Cursor] --> B[AI 聊天]
    A --> C[代码补全]
    A --> D[代码生成]
    A --> E[智能编辑]
    A --> F[调试辅助]
    
    B --> B1[上下文对话]
    B --> B2[代码问答]
    
    C --> C1[行内补全]
    C --> C2[函数补全]
    
    D --> D1[从注释生成]
    D --> D2[从描述生成]
    
    E --> E1[批量修改]
    E --> E2[代码重构]
```

### 与其他工具对比

| 特性 | Cursor | VS Code + Copilot | 传统 IDE |
|------|--------|------------------|---------|
| AI 集成度 | 深度集成 | 插件式 | 无/弱 |

# 多行补全
def process_data(data):
    # Cursor 可以生成完整函数体
    result = []
    for item in data:

            processed = item.transform()
            result.append(processed)
