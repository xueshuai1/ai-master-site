# 目标编码（Target Encoding）

## 1. 概述

目标编码是用**目标变量的统计量**（如均值）替换类别特征的编码方法，也称为均值编码。

**核心思想：** "用目标信息编码"——类别值用目标均值表示。

### 1.1 编码示例

```
原始数据：
类别    目标
A       1
A       0
B       1
B       1
B       0

目标编码：
A → (1+0)/2 = 0.5
B → (1+1+0)/3 = 0.67
```

### 1.2 优点

- 处理高基数类别
- 保留信息
- 维度不增加
- 树模型友好

### 1.3 风险

- 目标泄露
- 过拟合
- 需要平滑

## 2. Python 代码实现

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import KFold

def target_encode(train, test, col, target, k=1, global_mean=None):
    """带平滑的目标编码"""
    if global_mean is None:
        global_mean = train[target].mean()
    
    # 计算类别均值
    agg = train.groupby(col)[target].agg(['mean', 'count'])
    
    # 平滑
    lambda_factor = agg['count'] / (agg['count'] + k)
    smoothed = lambda_factor * agg['mean'] + (1 - lambda_factor) * global_mean
    
    # 应用编码
    train_encoded = train[col].map(smoothed)
    test_encoded = test[col].map(smoothed).fillna(global_mean)
    
    return train_encoded, test_encoded

# 示例
df = pd.DataFrame({
    'category': ['A', 'A', 'B', 'B', 'B', 'C'],
    'target': [1, 0, 1, 1, 0, 1]
})

encoded, _ = target_encode(df, df, 'category', 'target', k=1)
print(f"目标编码：{encoded.values}")
```

## 3. 总结

目标编码处理高基数类别：

**核心价值：**
1. 用目标信息编码
2. 处理高基数
3. 维度不增加
4. 需要防泄露

**最佳实践：**
- 使用交叉验证
- 添加平滑
- 防止过拟合
