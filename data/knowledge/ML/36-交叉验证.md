# 交叉验证（Cross-Validation）

## 1. 概述

交叉验证是通过**多次划分训练/测试集**来评估模型泛化能力的方法，是模型评估的金标准。

**核心思想：** "多次验证"——充分利用数据评估模型。

### 1.1 常用方法

| 方法 | 说明 | 适用 |
|------|------|------|
| K 折交叉验证 | 数据分 K 份 | 通用 |
| 分层 K 折 | 保持类别比例 | 分类 |
| 留一法 | 每次留 1 个 | 小数据 |
| 时间序列 | 按时间划分 | 时序数据 |

### 1.2 K 折交叉验证

```
数据分为 K 份
迭代 K 次：
  - K-1 份训练
  - 1 份测试
最终：K 次结果的平均
```

### 1.3 优点

- 充分利用数据
- 减少过拟合
- 评估稳定
- 模型选择

## 2. Python 代码实现

```python
import numpy as np
from sklearn.model_selection import (
    cross_val_score, KFold, StratifiedKFold,
    LeaveOneOut, TimeSeriesSplit
)
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target

# K 折交叉验证
kfold = KFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(RandomForestClassifier(), X, y, cv=kfold)
print(f"K 折 CV: {scores.mean():.4f} (+/- {scores.std()*2:.4f})")

# 分层 K 折（分类推荐）
skfold = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(RandomForestClassifier(), X, y, cv=skfold)
print(f"分层 K 折：{scores.mean():.4f}")

# 留一法（小数据）
loo = LeaveOneOut()
scores = cross_val_score(RandomForestClassifier(), X, y, cv=loo)
print(f"留一法：{scores.mean():.4f}")

# 时间序列
tscv = TimeSeriesSplit(n_splits=5)
```

## 3. 总结

交叉验证是模型评估标准：

**核心价值：**
1. 充分利用数据
2. 评估泛化能力
3. 模型选择
4. 减少过拟合

**最佳实践：**
- 分类用分层 K 折
- 时序用时间序列 CV
- 通常 K=5 或 10
- 报告均值和标准差
