# 分箱（Binning）

## 1. 概述

分箱是将**连续特征离散化**为区间的预处理方法，将连续值映射到有限的桶中。

**核心思想：** "化连续为离散"——将连续值分组。

### 1.1 分箱方法

| 方法 | 说明 | 示例 |
|------|------|------|
| 等宽分箱 | 等间隔 | [0-10], [10-20] |
| 等频分箱 | 等样本数 | 每箱 25% 样本 |
| 自定义分箱 | 业务定义 | 年龄：少年/青年/中年 |

### 1.2 优点

- 降低异常值影响
- 捕捉非线性
- 提高稳定性
- 增强可解释性

### 1.3 适用场景

- 异常值多
- 非线性关系
- 需要可解释性
- 树模型

## 2. Python 代码实现

```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import KBinsDiscretizer

# 生成数据
X = np.random.randn(100, 1) * 10 + 50

# 等宽分箱
kbin_width = KBinsDiscretizer(n_bins=5, encode='ordinal', strategy='uniform')
X_width = kbin_width.fit_transform(X)

# 等频分箱
kbin_freq = KBinsDiscretizer(n_bins=5, encode='ordinal', strategy='quantile')
X_freq = kbin_freq.fit_transform(X)

# pandas 实现
df = pd.DataFrame({'age': [25, 30, 35, 40, 45, 50, 55, 60]})
df['age_bin'] = pd.cut(df['age'], bins=[0, 30, 45, 60, 100], 
                       labels=['青年', '中年', '中老年', '老年'])
print(df)
```

## 3. 总结

分箱是离散化方法：

**核心价值：**
1. 连续转离散
2. 降低异常值影响
3. 捕捉非线性
4. 增强解释性

**适用场景：**
- 异常值多
- 非线性关系
- 需要可解释性
