# 特征缩放（Feature Scaling）

## 1. 概述

特征缩放是将特征缩放到**统一尺度**的预处理步骤，对距离-based 和梯度-based 算法至关重要。

**核心思想：** "统一尺度"——消除特征量纲影响。

### 1.1 常用方法

| 方法 | 公式 | 范围 | 适用 |
|------|------|------|------|
| 标准化 | (x-μ)/σ | 无限制 | 大多数算法 |
| 归一化 | (x-min)/(max-min) | [0,1] | 神经网络 |
| Robust | (x-median)/IQR | 无限制 | 有异常值 |

### 1.2 需要缩放的算法

- KNN、SVM、K-Means（距离-based）
- 神经网络、逻辑回归（梯度-based）
- PCA、LDA（方差-based）

### 1.3 不需要缩放的算法

- 决策树、随机森林
- XGBoost、LightGBM

## 2. Python 代码实现

```python
import numpy as np
from sklearn.preprocessing import StandardScaler, MinMaxScaler, RobustScaler

X = np.array([[1, 100], [2, 200], [3, 300], [4, 400]])

# 标准化
std_scaler = StandardScaler()
X_std = std_scaler.fit_transform(X)
print(f"标准化均值：{X_std.mean(axis=0)}")
print(f"标准化方差：{X_std.var(axis=0)}")

# 归一化
minmax_scaler = MinMaxScaler()
X_minmax = minmax_scaler.fit_transform(X)
print(f"归一化范围：[{X_minmax.min()}, {X_minmax.max()}]")

# Robust 缩放（抗异常值）
robust_scaler = RobustScaler()
X_robust = robust_scaler.fit_transform(X)
```

## 3. 总结

特征缩放是重要预处理：

**核心价值：**
1. 统一特征尺度
2. 加速收敛
3. 提高性能
4. 消除量纲

**最佳实践：**
- 训练集 fit，测试集 transform
- 标准化最常用
- 树模型不需要
