# 梯度下降（Gradient Descent）

## 1. 概述

梯度下降是**最优化算法**的基础，通过沿负梯度方向迭代更新参数，最小化损失函数。

**核心思想：** "下山法"——沿最陡方向下山。

### 1.1 算法公式

```
θ = θ - α × ∇L(θ)
```

其中：
- θ：参数
- α：学习率
- ∇L(θ)：损失函数梯度

### 1.2 变体

| 变体 | 说明 | 适用 |
|------|------|------|
| BG | 批量梯度下降 | 小数据 |
| SGD | 随机梯度下降 | 大数据 |
| Mini-batch | 小批量 | 通用 |

### 1.3 适用场景

- 线性回归
- 逻辑回归
- 神经网络
- 任何可微优化

## 2. Python 代码实现

```python
import numpy as np

def gradient_descent(X, y, learning_rate=0.01, n_iterations=1000):
    """批量梯度下降"""
    n_samples, n_features = X.shape
    theta = np.zeros(n_features)
    loss_history = []
    
    for _ in range(n_iterations):
        # 预测
        y_pred = X.dot(theta)
        
        # 计算梯度
        gradient = (1/n_samples) * X.T.dot(y_pred - y)
        
        # 更新参数
        theta -= learning_rate * gradient
        
        # 记录损失
        loss = np.mean((y - y_pred) ** 2)
        loss_history.append(loss)
    
    return theta, loss_history

# 测试
X = np.random.randn(100, 2)
y = X[:, 0] * 2 + X[:, 1] * 3 + np.random.randn(100) * 0.1

theta, losses = gradient_descent(X, y, learning_rate=0.1)
print(f"最优参数：{theta}")
```
