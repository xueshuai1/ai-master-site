# 学习率调度（Learning Rate Scheduling）

## 1. 概述

学习率调度是**动态调整学习率**的策略，在训练过程中根据进度或性能调整学习率，提高收敛速度和最终性能。

**核心思想：** "适时调整"——不同阶段用不同学习率。

### 1.1 常用调度策略

| 策略 | 公式 | 适用 |
|------|------|------|
| 时间衰减 | lr = lr₀ / (1 + kt) | 通用 |
| 指数衰减 | lr = lr₀ × e^(-kt) | 通用 |
| 阶梯衰减 | 定期减半 | 通用 |
| 余弦退火 | lr = lr₀ × (1+cos(πt/T))/2 | 深度学习 |
| 热身 | 先增后减 | Transformer |

### 1.2 为什么需要调度

- 初期：大学习率快速收敛
- 后期：小学习率精细调整
- 避免震荡
- 跳出局部最优

### 1.3 适用场景

- 深度学习
- 大规模训练
- 提高最终性能
- 稳定收敛

## 2. Python 代码实现

```python
import numpy as np
import matplotlib.pyplot as plt

def step_decay(epoch, initial_lr=0.1, drop=0.5, epochs_drop=10):
    """阶梯衰减"""
    return initial_lr * (drop ** (epoch // epochs_drop))

def exponential_decay(epoch, initial_lr=0.1, decay_rate=0.95):
    """指数衰减"""
    return initial_lr * (decay_rate ** epoch)

def cosine_annealing(epoch, initial_lr=0.1, total_epochs=100):
    """余弦退火"""
    return initial_lr * (1 + np.cos(np.pi * epoch / total_epochs)) / 2

def warmup_cosine(epoch, initial_lr=0.1, warmup_epochs=10, total_epochs=100):
    """热身 + 余弦退火"""
    if epoch < warmup_epochs:
        return initial_lr * (epoch + 1) / warmup_epochs
    else:
        return initial_lr * (1 + np.cos(np.pi * (epoch - warmup_epochs) / (total_epochs - warmup_epochs))) / 2

# 可视化
epochs = range(100)
lr_step = [step_decay(e) for e in epochs]
lr_exp = [exponential_decay(e) for e in epochs]
lr_cosine = [cosine_annealing(e, total_epochs=100) for e in epochs]
lr_warmup = [warmup_cosine(e, total_epochs=100) for e in epochs]

plt.figure(figsize=(12, 6))
plt.plot(epochs, lr_step, label='阶梯衰减')
plt.plot(epochs, lr_exp, label='指数衰减')
plt.plot(epochs, lr_cosine, label='余弦退火')
plt.plot(epochs, lr_warmup, label='热身 + 余弦')
plt.xlabel('Epoch')
plt.ylabel('Learning Rate')
plt.title('学习率调度策略')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

学习率调度提高训练效果：
