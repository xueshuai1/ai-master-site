# 标签编码（Label Encoding）

## 1. 概述

标签编码是将**类别标签转换为整数**的编码方法，每个类别对应一个唯一的整数值。

**核心思想：** "一个类别一个数字"——简单映射。

### 1.1 编码示例

```
原始：[红，绿，蓝，红]
标签编码：[0, 1, 2, 0]
```

### 1.2 与 One-Hot 对比

| 特性 | 标签编码 | One-Hot |
|------|----------|---------|
| 维度 | 1 | 类别数 |
| 顺序 | 隐含 | 无 |
| 适用 | 树模型 | 线性模型 |

### 1.3 适用场景

- 有序类别
- 树模型
- 类别数多
- 内存受限

## 2. Python 代码实现

```python
from sklearn.preprocessing import LabelEncoder
import pandas as pd

# 原始数据
data = ['红', '绿', '蓝', '红', '绿']

# 标签编码
le = LabelEncoder()
encoded = le.fit_transform(data)
print(f"编码：{encoded}")
print(f"类别：{le.classes_}")

# 逆转换
decoded = le.inverse_transform(encoded)
print(f"解码：{decoded}")

# pandas 实现
df = pd.DataFrame({'color': data})
df['color_encoded'] = df['color'].astype('category').cat.codes
print(df)
```

## 3. 总结

标签编码是简单编码方法：

**核心价值：**
1. 类别转整数
2. 维度低
3. 内存高效
4. 树模型友好

**适用场景：**
- 有序类别
- 树模型
- 类别数多
