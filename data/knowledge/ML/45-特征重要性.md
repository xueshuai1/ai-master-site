# 特征重要性（Feature Importance）

## 1. 概述

特征重要性衡量**每个特征对模型预测的贡献**，用于特征选择和模型解释。

**核心思想：** "特征贡献"——哪些特征最重要。

### 1.1 计算方法

| 方法 | 模型 | 说明 |
|------|------|------|
| 不纯度减少 | 树模型 | 分裂时不纯度减少 |
| 排列重要性 | 任意 | 打乱特征看性能下降 |
| 系数绝对值 | 线性 | 回归系数大小 |
| SHAP 值 | 任意 | 博弈论方法 |

### 1.2 适用场景

- 特征选择
- 模型解释
- 业务洞察
- 降维指导

## 2. Python 代码实现

```python
import numpy as np
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris
from sklearn.inspection import permutation_importance
import matplotlib.pyplot as plt

# 加载数据
iris = load_iris()
X, y = iris.data, iris.target
feature_names = iris.feature_names

# 训练模型
rf = RandomForestClassifier(n_estimators=100, random_state=42)
rf.fit(X, y)

# 1. 基于不纯度的重要性
importances = rf.feature_importances_
indices = np.argsort(importances)[::-1]

plt.figure(figsize=(10, 6))
plt.bar(range(X.shape[1]), importances[indices])
plt.xticks(range(X.shape[1]), np.array(feature_names)[indices], rotation=45)
plt.xlabel('特征')
plt.ylabel('重要性')
plt.title('特征重要性（不纯度）')
plt.tight_layout()
plt.show()

# 2. 排列重要性（更可靠）
perm_importance = permutation_importance(rf, X, y, n_repeats=10, random_state=42)
plt.figure(figsize=(10, 6))
plt.bar(range(X.shape[1]), perm_importance.importances_mean[indices])
plt.xticks(range(X.shape[1]), np.array(feature_names)[indices], rotation=45)
plt.xlabel('特征')
plt.ylabel('重要性')
plt.title('排列特征重要性')
plt.tight_layout()
plt.show()

print(f"Top 特征：{np.array(feature_names)[indices[0]]}")
```

## 3. 总结

特征重要性解释模型：

**核心价值：**
1. 量化特征贡献
2. 特征选择
3. 模型解释
4. 业务洞察

**最佳实践：**
- 排列重要性更可靠
- 结合多种方法
- 注意相关特征
- 考虑 SHAP 值
