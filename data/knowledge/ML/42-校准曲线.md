# 校准曲线（Calibration Curve）

## 1. 概述

校准曲线评估**预测概率的可靠性**，衡量预测概率与实际频率的一致性。

**核心思想：** "概率可信度"——预测 80% 时是否真的 80% 发生。

### 1.1 校准模型

- **完美校准**：预测概率 = 实际频率
- **过自信**：预测概率 > 实际频率
- **欠自信**：预测概率 < 实际频率

### 1.2 校准方法

| 方法 | 说明 | 适用 |
|------|------|------|
| Platt Scaling | Sigmoid 拟合 | 小数据 |
| Isotonic | 非参数 | 大数据 |

### 1.3 适用场景

- 需要可靠概率
- 风险评估
- 决策支持
- 模型比较

## 2. Python 代码实现

```python
from sklearn.calibration import calibration_curve, CalibratedClassifierCV
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import make_classification
import matplotlib.pyplot as plt

# 生成数据
X, y = make_classification(n_samples=1000, random_state=42)

# 训练模型
clf = RandomForestClassifier()
clf.fit(X, y)
y_proba = clf.predict_proba(X)[:, 1]

# 校准曲线
fraction_of_positives, mean_predicted_value = calibration_curve(y, y_proba, n_bins=10)

# 可视化
plt.figure(figsize=(10, 5))

plt.subplot(1, 2, 1)
plt.plot(mean_predicted_value, fraction_of_positives, 's-', label='模型')
plt.plot([0, 1], [0, 1], 'k--', label='完美校准')
plt.xlabel('平均预测概率')
plt.ylabel('正例比例')
plt.title('校准曲线')
plt.legend()
plt.grid(True, alpha=0.3)

plt.subplot(1, 2, 2)
plt.hist(y_proba, bins=10)
plt.xlabel('预测概率')
plt.ylabel('样本数')
plt.title('概率分布')
plt.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

# 校准模型
calibrated_clf = CalibratedClassifierCV(clf, method='sigmoid', cv=5)
calibrated_clf.fit(X, y)
```

## 3. 总结

校准曲线评估概率质量：

**核心价值：**
1. 评估概率可靠性
2. 校准预测概率
3. 提高决策质量
4. 模型比较

**最佳实践：**
- 需要概率时校准
- Platt 适合小数据
- Isotonic 适合大数据
- 交叉验证校准
