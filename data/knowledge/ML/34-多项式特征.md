# 多项式特征（Polynomial Features）

## 1. 概述

多项式特征是通过**生成原始特征的多项式组合**来扩展特征空间的方法，使线性模型能够拟合非线性关系。

**核心思想：** "升维拟合非线性"——用高维线性拟合低维非线性。

### 1.1 特征生成

```
原始：[x₁, x₂]
degree=2: [1, x₁, x₂, x₁², x₁x₂, x₂²]
degree=3: [1, x₁, x₂, x₁², x₁x₂, x₂², x₁³, x₁²x₂, x₁x₂², x₂³]
```

### 1.2 维度增长

```
原始维度：n
多项式维度：C(n+d, d) = (n+d)!/(n!d!)
```

### 1.3 适用场景

- 非线性关系
- 线性模型扩展
- 特征交互
- 小数据集

## 2. Python 代码实现

```python
import numpy as np
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import Pipeline

# 生成非线性数据
X = np.linspace(-5, 5, 100).reshape(-1, 1)
y = X**2 + np.random.randn(100, 1) * 2

# 多项式回归
model = Pipeline([
    ('poly', PolynomialFeatures(degree=2, include_bias=False)),
    ('linear', LinearRegression())
])

model.fit(X, y)
y_pred = model.predict(X)

# 评估
r2 = model.score(X, y)
print(f"R²: {r2:.4f}")
```

## 3. 总结

多项式特征扩展特征空间：

**核心价值：**
1. 生成多项式组合
2. 线性模型非线性化
3. 捕捉交互
4. 简单有效

**最佳实践：**
- 控制 degree（2-3 足够）
- 配合正则化
- 注意维度爆炸
