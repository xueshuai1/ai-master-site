# 特征提取（Feature Extraction）

## 1. 概述

特征提取是通过**变换或组合**原始特征，生成新的更有意义的特征表示。

**核心思想：** "创造更好特征"——从原始数据中提取有用信息。

### 1.1 常见方法

| 方法 | 说明 | 示例 |
|------|------|------|
| 多项式特征 | 特征组合 | x², xy, y² |
| PCA | 线性组合 | 主成分 |
| 文本特征 | 文本向量化 | TF-IDF |
| 图像特征 | 图像描述 | HOG, SIFT |

### 1.2 与特征选择对比

| 特性 | 特征选择 | 特征提取 |
|------|----------|----------|
| 特征数 | 减少 | 可变 |
| 原始特征 | 保留 | 变换 |
| 可解释性 | 高 | 低 |

## 2. Python 代码实现

```python
import numpy as np
from sklearn.preprocessing import PolynomialFeatures
from sklearn.decomposition import PCA
from sklearn.feature_extraction.text import TfidfVectorizer

# 1. 多项式特征
X = np.array([[1, 2], [3, 4]])
poly = PolynomialFeatures(degree=2)
X_poly = poly.fit_transform(X)
print(f"多项式特征：{X_poly.shape}")

# 2. PCA 特征提取
X_large = np.random.randn(100, 20)
pca = PCA(n_components=5)
X_pca = pca.fit_transform(X_large)
print(f"PCA 特征：{X_pca.shape}")

# 3. 文本特征提取
texts = ["hello world", "hello python", "goodbye world"]
vectorizer = TfidfVectorizer()
X_text = vectorizer.fit_transform(texts)
print(f"文本特征：{X_text.shape}")
```

## 3. 总结

特征提取创造新特征：

**核心价值：**
1. 变换原始特征
2. 提取有用信息
3. 提高模型能力
4. 适应不同数据

**适用场景：**
- 文本处理
- 图像处理
- 特征工程
- 降维
