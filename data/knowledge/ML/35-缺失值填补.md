# 缺失值填补（Missing Value Imputation）

## 1. 概述

缺失值填补是用**合理值替换缺失数据**的预处理方法，是数据清洗的关键步骤。

**核心思想：** "补全数据"——用估计值填充缺失。

### 1.1 缺失类型

| 类型 | 说明 | 处理 |
|------|------|------|
| MCAR | 完全随机缺失 | 直接删除或填补 |
| MAR | 随机缺失 | 模型填补 |
| MNAR | 非随机缺失 | 特殊处理 |

### 1.2 填补方法

| 方法 | 适用 | 说明 |
|------|------|------|
| 删除 | 缺失少 | 删除行/列 |
| 均值/中位数 | 数值特征 | 简单快速 |
| 众数 | 类别特征 | 最常见值 |
| 模型预测 | 复杂 | KNN、回归 |
| 多重填补 | 严格 | MICE |

## 2. Python 代码实现

```python
import numpy as np
import pandas as pd
from sklearn.impute import SimpleImputer, KNNImputer

# 创建含缺失的数据
df = pd.DataFrame({
    'A': [1, 2, np.nan, 4, 5],
    'B': [5, np.nan, np.nan, 8, 10],
    'C': ['a', 'b', np.nan, 'd', 'e']
})

# 均值填补
mean_imputer = SimpleImputer(strategy='mean')
df_mean = mean_imputer.fit_transform(df[['A', 'B']])

# 中位数填补
median_imputer = SimpleImputer(strategy='median')
df_median = median_imputer.fit_transform(df[['A', 'B']])

# 众数填补
most_frequent_imputer = SimpleImputer(strategy='most_frequent')
df_mode = most_frequent_imputer.fit_transform(df[['C']])

# KNN 填补
knn_imputer = KNNImputer(n_neighbors=2)
df_knn = knn_imputer.fit_transform(df[['A', 'B']])

# pandas 简单填补
df['A_filled'] = df['A'].fillna(df['A'].mean())
df['B_filled'] = df['B'].fillna(df['B'].median())
```

## 3. 总结

缺失值填补是数据预处理关键：

**核心价值：**
1. 处理缺失数据
2. 保留样本
3. 提高模型性能
4. 多种方法选择

**最佳实践：**
- 分析缺失模式
- 选择合适方法
- 避免数据泄露
- 考虑多重填补
