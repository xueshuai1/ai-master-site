# 交互特征（Interaction Features）

## 1. 概述

交互特征是通过**组合多个原始特征**创建的新特征，用于捕捉特征间的交互作用。

**核心思想：** "1+1>2"——特征组合产生新信息。

### 1.1 常见交互

| 类型 | 示例 |
|------|------|
| 乘法 | 面积 = 长 × 宽 |
| 除法 | 单价 = 总价 / 面积 |
| 多项式 | x², xy, y² |

### 1.2 优点

- 捕捉交互作用
- 提高模型能力
- 线性模型非线性化

### 1.3 缺点

- 维度增加
- 可能过拟合
- 计算成本

## 2. Python 代码实现

```python
import numpy as np
from sklearn.preprocessing import PolynomialFeatures

# 原始特征
X = np.array([[1, 2], [3, 4], [5, 6]])

# 多项式交互特征
poly = PolynomialFeatures(degree=2, include_bias=False)
X_poly = poly.fit_transform(X)
print(f"原始形状：{X.shape}")
print(f"交互后形状：{X_poly.shape}")
print(f"特征名：{poly.get_feature_names_out()}")
```

## 3. 总结

交互特征捕捉特征间关系：

**核心价值：**
1. 特征组合
2. 捕捉交互
3. 提高性能
4. 线性模型非线性化

**适用场景：**
- 特征有交互
- 线性模型
- 领域知识指导
