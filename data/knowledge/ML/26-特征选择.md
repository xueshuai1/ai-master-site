# 特征选择（Feature Selection）

## 1. 概述

特征选择是从原始特征中选择**最相关特征子集**的过程，用于减少维度、提高模型性能、降低过拟合风险。

**核心思想：** "少即是多"——选择最有信息量的特征。

### 1.1 方法分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 过滤法 | 基于统计指标 | 方差、相关性 |
| 包裹法 | 基于模型性能 | RFE |
| 嵌入法 | 模型内置选择 | L1 正则化 |

### 1.2 适用场景

- 高维数据
- 降低过拟合
- 提高训练速度
- 增强可解释性

## 2. Python 代码实现

```python
import numpy as np
from sklearn.feature_selection import (
    VarianceThreshold, SelectKBest, f_classif,
    RFE, SelectFromModel
)
from sklearn.ensemble import RandomForestClassifier
from sklearn.linear_model import Lasso
from sklearn.datasets import make_classification

# 生成数据
X, y = make_classification(n_samples=100, n_features=20, n_informative=10)

# 1. 方差选择
vt = VarianceThreshold(threshold=0.1)
X_vt = vt.fit_transform(X)

# 2. 单变量选择
skb = SelectKBest(score_func=f_classif, k=10)
X_skb = skb.fit_transform(X, y)

# 3. 递归特征消除
rfe = RFE(estimator=RandomForestClassifier(), n_features_to_select=10)
X_rfe = rfe.fit_transform(X, y)

# 4. 模型选择
sfm = SelectFromModel(Lasso(alpha=0.1))
X_sfm = sfm.fit_transform(X, y)

print(f"原始特征数：{X.shape[1]}")
print(f"方差选择：{X_vt.shape[1]}")
print(f"单变量选择：{X_skb.shape[1]}")
print(f"RFE: {X_rfe.shape[1]}")
print(f"模型选择：{X_sfm.shape[1]}")
```

## 3. 优缺点

**优点：**
- 降低维度
- 提高性能
- 减少过拟合
- 增强可解释性

**缺点：**
- 可能丢失信息
- 计算成本
- 需要验证

## 4. 总结

特征选择是重要预处理步骤：

**核心价值：**
1. 选择相关特征
2. 降低维度
3. 提高性能
4. 增强解释性

**适用场景：**
- 高维数据
- 过拟合风险
- 需要解释性
