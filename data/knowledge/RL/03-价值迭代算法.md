# 价值迭代算法

## 1. 概述

价值迭代（Value Iteration, VI）是强化学习中最经典的动态规划算法之一，用于求解马尔可夫决策过程（MDP）的最优策略。它通过迭代应用 Bellman 最优方程，逐步逼近最优价值函数，最终提取出最优策略。

**核心思想**：反复用 Bellman 最优方程更新状态价值，直到收敛到最优价值函数 V*。

价值迭代的特点：
- **无模型**：需要知道环境的完整模型（P 和 R）
- **离线规划**：在实际执行前计算最优策略
- **保证收敛**：对于有限 MDP 保证收敛到最优解
- **简单高效**：实现简单，适合中小规模问题

### 1.1 历史背景

价值迭代由 Richard Bellman 在 1957 年提出，是动态规划在序贯决策问题中的直接应用。它与策略迭代（Policy Iteration）并列为求解 MDP 的两大经典算法。

### 1.2 与策略迭代的区别

| 方面 | 价值迭代 | 策略迭代 |
|------|----------|----------|
| 更新对象 | 价值函数 V(s) | 策略π和价值 V^π |
| 每轮迭代 | 一次 Bellman 更新 | 完整策略评估 + 改进 |
| 收敛速度 | 较慢（线性） | 较快（迭代少） |
| 每轮成本 | 低 | 高（需解线性方程组） |
| 适用场景 | 大规模状态空间 | 中小规模 |

### 1.3 应用场景

- 网格世界路径规划
- 棋盘游戏 AI
- 资源调度优化
- 机器人导航
- 任何已知模型的 MDP 问题

## 2. 算法原理

### 2.1 Bellman 最优方程回顾

价值迭代基于 Bellman 最优方程：

```
V*(s) = max_a Σ_{s'} P(s'|s,a) [R(s,a,s') + γ V*(s')]
```

这个方程表明：最优状态价值等于所有可能动作中，能获得最大期望回报的那个动作的价值。

### 2.2 迭代更新规则

价值迭代将 Bellman 最优方程转化为迭代更新：

```
V_{k+1}(s) = max_a Σ_{s'} P(s'|s,a) [R(s,a,s') + γ V_k(s')]
```

其中 k 是迭代次数。

**关键性质**：
- 这个更新定义了一个算子 T：V_{k+1} = T(V_k)
- T 是γ-收缩映射：||T(V1) - T(V2)||_∞ ≤ γ ||V1 - V2||_∞
- 根据 Banach 不动点定理，T 有唯一不动点 V*
- 迭代收敛：lim_{k→∞} V_k = V*

### 2.3 收敛性分析

**收敛速度**：
```
||V_k - V*||_∞ ≤ γ^k ||V_0 - V*||_∞
```

- 线性收敛，收敛率由γ决定
- γ越接近 1，收敛越慢
- 通常γ=0.9 或 0.99

**停止条件**：
```
max_s |V_{k+1}(s) - V_k(s)| < ε
```

其中ε是预设的精度阈值（如 1e-6）。

### 2.4 最优策略提取

收敛后，从最优价值函数提取最优策略：

```
π*(s) = argmax_a Σ_{s'} P(s'|s,a) [R(s,a,s') + γ V*(s')]
```

这等价于：
```
π*(s) = argmax_a Q*(s,a)
```

其中 Q*(s,a) = Σ_{s'} P(s'|s,a) [R(s,a,s') + γ V*(s')]

## 3. 算法流程

### 3.1 标准价值迭代

```mermaid
flowchart TD
    Start([开始]) --> Init[初始化 V(s)=0, ∀s]
    Init --> SetDelta[Δ = 0]
    SetDelta --> LoopS[遍历每个状态 s]
    LoopS --> CalcQ[计算 Q(s,a) = Σ P(s'|s,a)[R + γV(s')]]
    CalcQ --> MaxQ[V_new(s) = max_a Q(s,a)]
    MaxQ --> UpdateDelta[Δ = max(Δ, |V_new(s)-V(s)|)]
    UpdateDelta --> NextS{还有状态？}
    NextS -->|是 | LoopS
    NextS -->|否 | UpdateV[V ← V_new]
    UpdateV --> Check{Δ < ε?}
    Check -->|否 | SetDelta
    Check -->|是 | Extract[提取π*(s) = argmax_a Q(s,a)]
    Extract --> End([输出 V*, π*])

```

### 3.2 伪代码

```
算法：价值迭代（Value Iteration）

输入：
    MDP = (S, A, P, R, γ)
    精度阈值ε

输出：
    最优价值函数 V*
    最优策略π*

步骤：
1. 初始化：V(s) = 0, ∀s ∈ S
2. 重复：
    a. Δ ← 0
    b. 对于每个状态 s ∈ S：
        i.   v ← V(s)
        ii.  对于每个动作 a ∈ A：
                 Q(s,a) ← Σ_{s'} P(s'|s,a) [R(s,a,s') + γ V(s')]
        iii. V(s) ← max_a Q(s,a)
        iv.  Δ ← max(Δ, |v - V(s)|)
    c. 如果 Δ < ε，跳出循环
3. 提取最优策略：
    对于每个状态 s ∈ S：
        π*(s) ← argmax_a Σ_{s'} P(s'|s,a) [R(s,a,s') + γ V(s')]
4. 返回 V, π*
```

### 3.3 原地更新 vs 同步更新

**同步更新**（标准 VI）：
```python
V_new = np.zeros(n_states)
for s in range(n_states):
    V_new[s] = max_a Q(s, a, V)  # 使用旧 V
V = V_new
```

**原地更新**（Gauss-Seidel 风格）：
```python
for s in range(n_states):
    V[s] = max_a Q(s, a, V)  # 使用部分更新的 V
```

原地更新通常收敛更快，因为利用了最新的信息。

## 4. 代码实现

```python
import numpy as np

class ValueIteration:
    """价值迭代算法实现"""
    
    def __init__(self, n_states, n_actions, P, R, gamma=0.9):
        """
        参数:
            n_states: 状态数量
            n_actions: 动作数量
            P: 状态转移概率 P[s][a][s']
            R: 奖励函数 R[s][a][s']
            gamma: 折扣因子
        """
        self.n_states = n_states
        self.n_actions = n_actions
        self.P = np.array(P)
        self.R = np.array(R)
        self.gamma = gamma
    
    def solve(self, threshold=1e-6, max_iterations=10000):
        """
        执行价值迭代求解最优策略
        
        返回:
            V: 最优价值函数
            policy: 最优策略
            iterations: 迭代次数
        """
        # 初始化价值函数
        V = np.zeros(self.n_states)
        
        for iteration in range(max_iterations):
            delta = 0
            V_new = np.zeros(self.n_states)
            
            # 遍历所有状态
            for s in range(self.n_states):
                # 计算每个动作的 Q 值
                q_values = np.zeros(self.n_actions)
                
                for a in range(self.n_actions):
                    # Q(s,a) = Σ P(s'|s,a) [R(s,a,s') + γ V(s')]
                    q_values[a] = np.sum(
                        self.P[s, a] * (self.R[s, a] + self.gamma * V)
                    )
                
                # 取最大值
                V_new[s] = np.max(q_values)
                
                # 跟踪最大变化
                delta = max(delta, abs(V_new[s] - V[s]))
            
            # 更新价值函数
            V = V_new
            
            # 检查收敛
            if delta < threshold:
                print(f"价值迭代在 {iteration + 1} 次迭代后收敛")
                break
        
        # 提取最优策略
        policy = np.zeros(self.n_states, dtype=int)
        Q_optimal = np.zeros((self.n_states, self.n_actions))
        
        for s in range(self.n_states):
            for a in range(self.n_actions):
                Q_optimal[s, a] = np.sum(
                    self.P[s, a] * (self.R[s, a] + self.gamma * V)
                )
            policy[s] = np.argmax(Q_optimal[s])
        
        return V, policy, iteration + 1, Q_optimal
    
    def evaluate_policy(self, policy, threshold=1e-6):
        """
        评估给定策略的价值函数（策略评估）
        """
        V = np.zeros(self.n_states)
        
        for _ in range(10000):
            delta = 0
            for s in range(self.n_states):
                v = V[s]
                a = policy[s]
                V[s] = np.sum(
                    self.P[s, a] * (self.R[s, a] + self.gamma * V)
                )
                delta = max(delta, abs(v - V[s]))
            
            if delta < threshold:
                break
        
        return V

# 示例：FrozenLake 环境
def create_frozen_lake(size=4):
    """
    创建 FrozenLake 风格的网格世界
    
    S: 起点，G: 终点，F: 冰面，H: 洞
    """
    n_states = size * size
    n_actions = 4  # 左、下、右、上
    
    P = np.zeros((n_states, n_actions, n_states))
    R = np.zeros((n_states, n_actions, n_states))
    
    # 定义动作位移
    actions = [(0, -1), (1, 0), (0, 1), (-1, 0)]
    
    # 设置洞和终点（示例）
    holes = {5, 7, 11, 12}  # 状态索引
    goal = 15
    
    for s in range(n_states):
        row, col = s // size, s % size
        
        for a in range(n_actions):
            dr, dc = actions[a]
            new_row = max(0, min(size-1, row + dr))
            new_col = max(0, min(size-1, col + dc))
            s_next = new_row * size + new_col
            
            if s_next in holes:
                # 掉入洞：留在原地，负奖励
                P[s][a][s] = 1.0
                R[s][a][s] = -1
            elif s_next == goal:
                # 到达终点
                P[s][a][s_next] = 1.0
                R[s][a][s_next] = 1
            else:
                # 正常移动
                P[s][a][s_next] = 1.0
                R[s][a][s_next] = -0.01  # 每步小惩罚
    
    return n_states, n_actions, P, R

# 运行示例
if __name__ == "__main__":
    # 创建环境
    n_states, n_actions, P, R = create_frozen_lake(4)
    
    # 创建求解器
    vi = ValueIteration(n_states, n_actions, P, R, gamma=0.95)
    
    # 求解
    V, policy, iterations, Q = vi.solve(threshold=1e-6)
    
    # 显示结果
    print(f"\n迭代次数：{iterations}")
    print("\n最优价值函数:")
    print(V.reshape(4, 4))
    
    print("\n最优策略 (0:左，1:下，2:右，3:上):")
    print(policy.reshape(4, 4))
    
    # 验证策略
    V_eval = vi.evaluate_policy(policy)
    print("\n策略评估验证:")
    print(V_eval.reshape(4, 4))
```

## 5. 应用场景

### 5.1 网格世界导航

在机器人导航中，价值迭代可以计算从任意位置到目标的最优路径：
- 状态：网格位置
- 动作：移动方向
- 奖励：到达目标 +1，碰撞 -1，每步 -0.01

### 5.2 游戏 AI

在棋类游戏中：
- 状态：棋盘配置
- 动作：合法走法
- 奖励：赢 +1，输 -1，平局 0

### 5.3 资源调度

在云计算资源管理中：
- 状态：当前资源使用情况
- 动作：分配/释放资源
- 奖励：满足 SLA + 奖励，违反 - 惩罚

### 5.4 库存管理

在供应链优化中：
- 状态：当前库存水平
- 动作：订购数量
- 奖励：销售利润 - 库存成本

## 6. 算法变体

### 6.1 异步价值迭代

不遍历所有状态，而是按特定顺序更新：

```
重复：
    选择状态 s（按某种策略）
    V(s) ← max_a Σ P(s'|s,a) [R + γ V(s')]
```

优点：可以优先更新重要的状态。

### 6.2 实时动态规划（RTDP）

结合模拟的价值迭代：

```
从当前状态 s 开始：
    重复：
        a ← argmax_a Σ P(s'|s,a) [R + γ V(s')]
        采样下一状态 s' ∼ P(·|s,a)
        V(s) ← Σ P(s'|s,a) [R + γ V(s')]
        s ← s'
```

### 6.3 优先扫描（Prioritized Sweeping）

维护优先队列，优先更新变化大的状态：

```
初始化优先队列 PQ
当 PQ 非空：
    s ← PQ.pop()  # 优先级最高的状态
    V(s) ← max_a Σ P(s'|s,a) [R + γ V(s')]
    对于每个前驱状态 s'：
        如果 |V(s') - 旧值| 大：
            PQ.push(s')
```

## 7. 总结

价值迭代是求解 MDP 的基础算法：

1. **理论基础**：基于 Bellman 最优方程
2. **收敛保证**：对有限 MDP 保证收敛到最优
3. **实现简单**：核心是 max 和期望的迭代
4. **适用性**：需要完整模型，适合规划问题
5. **扩展性**：有多种变体适应不同场景

虽然价值迭代需要环境模型，但它的思想启发了许多无模型算法（如 Q-Learning）。
