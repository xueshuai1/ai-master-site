# 价值迭代算法面试题（5 道）

## 题目 1：基础概念
**问题：** 价值迭代算法的核心思想是什么？它与 Bellman 最优方程有什么关系？

**参考答案：**
**核心思想**：
- 通过迭代应用 Bellman 最优方程，逐步逼近最优价值函数 V*
- 每次迭代用当前价值估计更新所有状态的价值
- 收敛后从 V* 提取最优策略

**与 Bellman 最优方程的关系**：
- Bellman 最优方程：V*(s) = max_a Σ P(s'|s,a)[R(s,a,s') + γV*(s')]
- 价值迭代将其转化为迭代格式：V_{k+1}(s) = max_a Σ P(s'|s,a)[R(s,a,s') + γV_k(s')]
- 价值迭代是求解 Bellman 最优方程的数值方法

**关键点**：
- 定义 Bellman 算子 T：V_{k+1} = T(V_k)
- T 是收缩映射，保证收敛到唯一不动点 V*

**评分要点：** 说出迭代逼近得基本分，提到收缩映射得高分

## 题目 2：收敛性分析
**问题：** 价值迭代为什么能保证收敛？收敛速度如何？

**参考答案：**
**收敛保证**：
1. **Bellman 算子是收缩映射**：
   ||T(V1) - T(V2)||_∞ ≤ γ ||V1 - V2||_∞
   
2. **Banach 不动点定理**：
   - 收缩映射在完备度量空间有唯一不动点
   - 迭代序列收敛到该不动点

3. **条件**：
   - 有限状态空间
   - γ < 1

**收敛速度**：
- **线性收敛**：||V_k - V*||_∞ ≤ γ^k ||V_0 - V*||_∞
- **收敛率**：由γ决定
  - γ=0.9：每迭代 1 次误差乘以 0.9
  - γ=0.99：收敛较慢
- **迭代次数估计**：
  要达到精度ε，需要 k ≥ log(ε/||V_0-V*||_∞) / log(γ)

**评分要点：** 说出收缩映射得基本分，给出收敛率公式得高分

## 题目 3：算法实现
**问题：** 请写出价值迭代的核心更新代码，并解释如何提取最优策略。

**参考答案：**
```python
import numpy as np

def value_iteration(P, R, gamma, threshold=1e-6):
    """
    P: 状态转移概率 P[s][a][s']
    R: 奖励函数 R[s][a][s']
    """
    n_states, n_actions = P.shape[0], P.shape[1]
    V = np.zeros(n_states)
    
    # 价值迭代
    while True:
        delta = 0
        V_new = np.zeros(n_states)
        
        for s in range(n_states):
            # 计算每个动作的 Q 值
            q_values = np.zeros(n_actions)
            for a in range(n_actions):
                q_values[a] = np.sum(P[s][a] * (R[s][a] + gamma * V))
            
            # 取最大值
            V_new[s] = np.max(q_values)
            delta = max(delta, abs(V_new[s] - V[s]))
        
        V = V_new
        
        if delta < threshold:
            break
    
    # 提取最优策略
    policy = np.zeros(n_states, dtype=int)
    for s in range(n_states):
        q_values = np.zeros(n_actions)
        for a in range(n_actions):
            q_values[a] = np.sum(P[s][a] * (R[s][a] + gamma * V))
        policy[s] = np.argmax(q_values)
    
    return V, policy
```

**策略提取**：
```
π*(s) = argmax_a Q*(s,a)
      = argmax_a Σ P(s'|s,a)[R(s,a,s') + γV*(s')]
```

**评分要点：** 代码正确得基本分，解释 argmax 含义得高分

## 题目 4：算法对比
**问题：** 比较价值迭代和策略迭代的异同，各自适用什么场景？

**参考答案：**

| 方面 | 价值迭代 | 策略迭代 |
|------|----------|----------|
| **更新对象** | 只更新 V(s) | 交替更新 V^π和π |
| **迭代结构** | 单一循环 | 双重循环（评估 + 改进） |
| **每轮成本** | O(|S|²|A|) | O(|S|³)（需解方程） |
| **迭代次数** | 多 | 少（通常<10） |
| **收敛速度** | 线性 | 二次（更快） |
| **内存需求** | 低 | 高 |

**相同点**：
- 都基于 Bellman 方程
- 都需要完整模型（P, R）
- 都保证收敛到最优策略

**适用场景**：
- **价值迭代**：
  - 状态空间大
  - 每轮迭代成本敏感
  - 实现简单优先

- **策略迭代**：
  - 状态空间中小
  - 需要快速收敛
  - 可以承受高计算成本

**实际选择**：
- 大规模问题：价值迭代或其变体
- 中小规模：策略迭代
- 现代应用：两者结合（修改策略迭代）

**评分要点：** 说出更新对象区别得基本分，详细对比表格得高分

## 题目 5：应用分析
**问题：** 在一个 3x3 网格中，起点 (0,0)，终点 (2,2) 奖励 +10，每步 -1。使用价值迭代：
1. 写出状态转移和奖励定义
2. 计算第一次迭代后终点的价值
3. 解释折扣因子γ=0.9 的影响

**参考答案：**
**1. 状态转移和奖励**：
- 状态：s = (row, col)，共 9 个状态
- 动作：上、下、左、右
- 转移：确定性，P(s'|s,a) = 1（如果移动合法）
- 奖励：
  - R(s, a, 终点) = +10
  - R(s, a, 其他) = -1

**2. 第一次迭代后终点价值**：
- 初始：V₀(s) = 0, ∀s
- 终点状态 s_goal = (2,2)：
  - 任何动作都留在终点（吸收态）
  - V₁(s_goal) = max_a [R + γ·V₀(s_goal)] = 10 + 0.9×0 = 10

**3. γ=0.9 的影响**：
- **价值衰减**：距离终点 k 步的状态价值 ≈ 10×0.9^k - 步数成本
- **收敛速度**：γ=0.9 时收敛适中
- **长期规划**：γ接近 1 更重视未来，γ小更短视

**具体计算示例**（距离终点 2 步的状态）：
```
V₁(距离 2 步) = -1 + 0.9×V₀(距离 1 步) = -1
V₂(距离 2 步) = -1 + 0.9×V₁(距离 1 步) = -1 + 0.9×(-1+0.9×10) = -1 + 0.9×8 = 6.2
```

**评分要点：** 正确定义 MDP 得基本分，准确计算得高分
