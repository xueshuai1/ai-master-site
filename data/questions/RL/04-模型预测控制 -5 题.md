## 题目 1：基础概念
**问题：** 什么是模型预测控制（MPC）？它的核心思想是什么？

**参考答案：**
**模型预测控制（MPC）**：
- 一种基于模型的优化控制方法
- 在每个时间步求解有限时域的最优控制问题
- 只执行第一个控制动作，然后重新优化

**核心思想**：
1. **预测**：使用系统模型预测未来行为
2. **优化**：求解满足约束的最优控制序列
3. **滚动**：执行一步后重新测量、重新优化
4. **反馈**：通过重优化实现闭环控制

**关键特点**：
- 显式处理约束（状态、输入）
- 适用于多变量系统
- 在线计算，计算负担较大

**评分要点：** 说出"预测 + 优化 + 滚动"得基本分，提到显式约束得高分

## 题目 2：MPC 与 RL 对比
**问题：** MPC 和强化学习有什么异同？它们可以如何结合？

**参考答案：**

| 方面 | MPC | 强化学习 |
|------|-----|----------|
| **模型** | 需要显式模型 | 可以有/无模型 |
| **优化** | 在线优化 | 离线学习 |
| **约束** | 显式处理 | 通过奖励隐式 |
| **计算** | 在线计算密集 | 离线训练，在线快速 |
| **理论保证** | 稳定性可证明 | 收敛性依赖算法 |
| **适用** | 已知模型、约束重要 | 模型未知、复杂任务 |

**相同点**：
- 都是序贯决策方法
- 都考虑长期回报/成本
- 都可以处理连续控制

**结合方式**：
1. **学习模型 + MPC**：
   - 用 RL/监督学习学习系统模型
   - 用 MPC 进行规划控制
   
2. **MPC 作为策略**：
   - 将 MPC 视为参数化策略
   - 用 RL 优化 MPC 参数（权重、时域）

3. **RL 提供初始解**：
   - RL 提供 warm start
   - MPC 进行精细优化

4. **分层架构**：
   - 高层：RL 做决策
   - 低层：MPC 做跟踪控制

**评分要点：** 说出主要区别得基本分，提出结合方式得高分

## 题目 3：优化问题
**问题：** 写出线性二次型 MPC 的优化问题公式，并解释各项含义。

**参考答案：**
**优化问题**：
```
min_{u_0,...,u_{N-1}} J = Σ_{k=0}^{N-1} (x_k^T Q x_k + u_k^T R u_k) + x_N^T P x_N

约束条件:
    x_0 = x_t（当前状态）
    x_{k+1} = A x_k + B u_k（系统动态）
    u_min ≤ u_k ≤ u_max（输入约束）
    x_min ≤ x_k ≤ x_max（状态约束）
```

**各项含义**：
- **J**：总成本函数
- **x_k^T Q x_k**：状态成本（跟踪误差）
  - Q：状态权重矩阵（正定/半正定）
  - 惩罚偏离参考状态
- **u_k^T R u_k**：控制成本（能量/平滑）
  - R：控制权重矩阵（正定）
  - 惩罚过大控制输入
- **x_N^T P x_N**：终端成本
  - P：终端权重
  - 保证稳定性
- **N**：预测时域

**权重选择**：
- Q 大：重视跟踪精度
- R 大：重视控制平滑/节能
- N 大：更长视野，更稳定

**评分要点：** 写出完整公式得基本分，解释权重含义得高分

## 题目 4：代码实现
**问题：** 请实现 MPC 的预测函数，给定初始状态和控制序列，计算状态轨迹。

**参考答案：**
```python
import numpy as np

class MPC:
    def __init__(self, A, B, N):
        """
        A: 状态转移矩阵 (nx x nx)
        B: 控制矩阵 (nx x nu)
        N: 预测时域
        """
        self.A = A
        self.B = B
        self.N = N
        self.nx = A.shape[0]
        self.nu = B.shape[1]
    
    def predict(self, x0, U):
        """
        预测状态轨迹
        
        参数:
            x0: 初始状态 (nx,)
            U: 控制序列 [u_0, u_1, ..., u_{N-1}] (N*nu,)
        
        返回:
            X: 状态轨迹 [x_0, x_1, ..., x_N] ((N+1) x nx)
        """
        X = [x0.copy()]
        x = x0.copy()
        
        for k in range(self.N):
            # 提取第 k 步控制
            u = U[k * self.nu:(k + 1) * self.nu]
            
            # 状态更新：x_{k+1} = A x_k + B u_k
            x = self.A @ x + self.B @ u
            
            X.append(x.copy())
        
        return np.array(X)
    
    def step(self, x0, U_opt):
        """
        执行一步控制
        
        返回:
            u0: 要执行的控制动作
            x_next: 下一状态（预测）
        """
        u0 = U_opt[:self.nu]
        x_next = self.A @ x0 + self.B @ u0
        return u0, x_next

# 使用示例
if __name__ == "__main__":
    # 简单系统
    A = np.array([[1, 0.1], [0, 1]])
    B = np.array([[0], [0.1]])
    
    mpc = MPC(A, B, N=10)
    
    x0 = np.array([0, 0])
    U = np.zeros(10)  # 10 步控制
    
    X = mpc.predict(x0, U)
    print("状态轨迹:")
    print(X)
```

**关键点**：
1. 迭代应用 x_{k+1} = A x_k + B u_k
2. 返回包括初始状态的完整轨迹
3. 控制序列需要正确切片

**评分要点：** 代码正确实现动态得基本分，处理边界得高分

## 题目 5：稳定性分析
**问题：** MPC 如何保证闭环稳定性？有哪些方法？

**参考答案：**
**MPC 稳定性保证方法**：

**1. 终端约束法**：
```
添加约束：x_N ∈ X_f
```
- X_f 是控制不变集
- 保证存在可行控制使状态保持在 X_f
- 证明：价值函数是 Lyapunov 函数

**2. 终端成本法**：
```
添加终端成本：l_N(x) = x^T P x
```
- P 是无限时域 LQR 的价值矩阵
- 满足 Lyapunov 方程：P = Q + A^T P A - A^T P B (R + B^T P B)^{-1} B^T P A
- 即使没有终端约束也稳定

**3. 足够长时域**：
- N 足够大时，MPC 近似无限时域最优控制
- 无需终端约束/成本
- 但 N 需要很大

**4. 收缩约束**：
```
||x_N|| ≤ ρ ||x_0||, ρ < 1
```
- 强制状态在预测时域末端收缩

**稳定性证明思路**：
1. 定义价值函数 V(x) = 最优成本 J*(x)
2. 证明 V(x) 是正定的
3. 证明 V(x_{t+1}) - V(x_t) ≤ -l(x_t, u_t) < 0
4. 由 Lyapunov 稳定性定理，系统稳定

**评分要点：** 说出终端约束/成本得基本分，解释 Lyapunov 证明得高分
