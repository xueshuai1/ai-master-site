# 部分可观测 MDP 面试题（5 道）

## 题目 1：基础概念
**问题：** 什么是 POMDP？它与 MDP 的核心区别是什么？为什么 POMDP 更难求解？

**参考答案：**
**POMDP 定义**：
部分可观测马尔可夫决策过程，智能体无法直接观测到完整状态，只能通过观测推断。

**与 MDP 的核心区别**：

| 方面 | MDP | POMDP |
|------|-----|-------|
| 观测 | 完全可观测 s_t | 部分可观测 o_t |
| 决策依据 | 状态 s | 信念 b(s) |
| 马尔可夫性 | 状态满足 | 信念状态满足 |
| 策略 | π(s) → a | π(b) → a |

**POMDP 更难的原因**：
1. **信念空间连续**：即使 S 有限，信念空间是连续的
2. **计算复杂度**：精确求解是 PSPACE-完全（MDP 是 P-完全）
3. **信息价值**：动作有收集信息的价值，需权衡
4. **历史依赖**：需要维护整个历史或信念

**评分要点：** 说出信念状态得基本分，解释复杂度差异得高分

## 题目 2：信念更新
**问题：** 写出 POMDP 信念更新的贝叶斯公式，并解释每一步的含义。

**参考答案：**
**信念更新公式**：
```
b'(s') = P(s' | b, a, o)
       = η · P(o | s', a) · Σ_s P(s' | s, a) · b(s)
       = η · Z(o|s',a) · Σ_s T(s'|s,a) · b(s)
```

其中η是归一化常数：η = 1 / P(o|b,a)

**步骤解释**：

1. **预测步骤**（先验）：
   ```
   b_pred(s') = Σ_s T(s'|s,a) · b(s)
   ```
   - 根据动作预测下一状态分布
   - 忽略观测，只用转移模型

2. **更新步骤**（似然）：
   ```
   b'(s') ∝ Z(o|s',a) · b_pred(s')
   ```
   - 用观测修正预测
   - 观测可能性高的状态概率增加

3. **归一化**：
   ```
   b'(s') = b'(s') / Σ_s' b'(s')
   ```
   - 确保概率和为 1

**直观理解**：
- 预测：做了动作 a，可能到什么状态
- 更新：看到观测 o，哪些状态更可能
- 类似卡尔曼滤波的预测 - 更新循环

**评分要点：** 写出公式得基本分，解释预测 - 更新得高分

## 题目 3：信念空间 MDP
**问题：** 为什么说信念状态 b 满足马尔可夫性？如何将 POMDP 转化为信念空间 MDP？

**参考答案：**
**信念状态的马尔可夫性**：

给定当前信念 b_t，未来与过去独立：
```
P(b_{t+1} | b_t, a_t, b_{t-1}, ...) = P(b_{t+1} | b_t, a_t)
```

**原因**：
- 信念 b_t 包含了历史的所有相关信息
- b_t = P(s_t | h_t) 是状态的充分统计量
- 信念更新只依赖 (b_t, a_t, o_{t+1})

**转化为信念空间 MDP**：

```
信念 MDP = (B, A, τ, ρ, γ)

状态空间 B: 信念单纯形 Δ(S) = {b | Σ b(s)=1, b(s)≥0}
动作空间 A: 与原 POMDP 相同
转移函数 τ: τ(b'|b,a) = P(b'|b,a) = Σ_o P(b'|b,a,o)P(o|b,a)
奖励函数 ρ: ρ(b,a) = Σ_s b(s)R(s,a)（期望奖励）
折扣因子 γ: 不变
```

**价值函数**：
```
V*(b) = max_a [ρ(b,a) + γ Σ_o P(o|b,a) V*(b')]
```

**关键洞察**：
- 信念空间 MDP 是连续状态 MDP
- 最优价值函数是分段线性和凸的
- 可用α-向量表示：V(b) = max_{α∈Γ} α·b

**评分要点：** 说出充分统计量得基本分，写出转化公式得高分

## 题目 4：QMDP 算法
**问题：** 解释 QMDP 算法的原理，它是什么类型的近似？有什么优缺点？

**参考答案：**
**QMDP 原理**：

假设"一步之后完全可观测"，用 MDP 价值近似 POMDP：

```
Q_POMDP(b,a) ≈ Σ_s b(s) · Q_MDP(s,a)

其中 Q_MDP(s,a) = R(s,a) + γ Σ_{s'} T(s'|s,a) V_MDP(s')
```

**算法步骤**：
1. 求解底层 MDP 的最优价值 V_MDP
2. 计算 Q_MDP(s,a) 对所有 s,a
3. 对于信念 b，计算 Q(b,a) = Σ_s b(s)Q_MDP(s,a)
4. 选择动作：a* = argmax_a Q(b,a)

**近似类型**：
- **上界近似**（对最大化问题）
- 假设未来完全可观测，高估价值

**优点**：
- 计算简单高效
- 只需解一次 MDP
- 在线决策快速

**缺点**：
- 忽略信息收集价值
- 不会主动获取信息
- 在需要探索时表现差

**适用场景**：
- 观测信息量大
- 信息收集不重要
- 需要快速决策

**评分要点：** 说出"一步后完全可观测"得基本分，分析优缺点得高分

## 题目 5：应用分析
**问题：** 在老虎机问题（Tiger Problem）中：
- 2 状态：老虎在左/右
- 3 动作：开左、开右、听
- 听声音有 85% 准确率，代价 -1
- 开对门 +10，开错门 -100

1. 初始信念均匀分布，应该采取什么动作？
2. 如果听到左边有声音，信念如何更新？
3. 为什么不能直接开门？

**参考答案：**
**1. 初始动作选择**：

初始信念：b = [0.5, 0.5]

直接开门的期望价值：
- 开左：0.5×(-100) + 0.5×10 = -45
- 开右：0.5×10 + 0.5×(-100) = -45

听的期望价值：
- 代价 -1，但获得信息
- 之后可以以 85% 概率开对门

应该选择**听**，因为信息价值超过代价。

**2. 信念更新**：

听到左边有声音 (o=左)：
```
b'(老虎在左) ∝ Z(左 | 左) × b(左) = 0.85 × 0.5 = 0.425
b'(老虎在右) ∝ Z(左 | 右) × b(右) = 0.15 × 0.5 = 0.075

归一化：b' = [0.425/0.5, 0.075/0.5] = [0.85, 0.15]
```

**3. 为什么不能直接开门**：

- **期望损失大**：均匀信念下直接开门期望 -45
- **信息价值高**：听声音可以将正确率从 50% 提升到 85%
- **风险规避**：-100 的惩罚远大于 -1 的听代价
- **最优策略**：听几次直到信念足够确定，再开门

**量化分析**：
听一次后信念 [0.85, 0.15]：
- 开左期望：0.85×(-100) + 0.15×10 = -83.5
- 开右期望：0.85×10 + 0.15×(-100) = -6.5

听两次（都听到左）后信念约 [0.98, 0.02]：
- 开右期望：0.98×10 + 0.02×(-100) = 7.8 > 0

所以听两次后开右是正的期望！

**评分要点：** 正确计算期望得基本分，完整分析得高分
