# 线性回归面试题（5 道）

## 题目 1：基础概念
**问题：** 请解释线性回归的基本假设有哪些？如果这些假设被违反会产生什么后果？

**参考答案：**
线性回归有 5 个基本假设：
1. **线性关系**：自变量与因变量之间存在线性关系
2. **独立性**：残差之间相互独立（无自相关）
3. **同方差性**：残差具有恒定方差
4. **正态性**：残差服从正态分布
5. **无多重共线性**：自变量之间不存在高度相关

违反后果：
- 非线性→模型欠拟合，预测偏差
- 不独立→标准误估计错误
- 异方差→置信区间无效
- 非正态→假设检验失效
- 多重共线性→系数不稳定

**评分要点：** 说出 3 个以上假设得基本分，能说明后果得高分

## 题目 2：损失函数
**问题：** 为什么线性回归使用 MSE 作为损失函数？MSE 有什么优缺点？

**参考答案：**
**为什么用 MSE：**
1. 处处可导，便于梯度下降优化
2. 对大误差敏感，更好惩罚异常值
3. 高斯噪声假设下的最大似然估计
4. 有解析解（正规方程）

**优点：**
- 数学性质好（凸函数）
- 计算简单
- 统计意义明确

**缺点：**
- 对异常值敏感
- 假设误差正态分布

**评分要点：** 说出可导性和最大似然估计得高分

## 题目 3：正则化
**问题：** Ridge 回归和 Lasso 回归有什么区别？各自适用什么场景？

**参考答案：**
**区别：**
| 特性 | Ridge (L2) | Lasso (L1) |
|------|-----------|-----------|
| 正则项 | Σwᵢ² | Σ|wᵢ| |
| 系数 | 缩小不为零 | 可压缩为零 |
| 特征选择 | 无 | 有 |
| 解析解 | 有 | 无 |

**适用场景：**
- Ridge：特征多且都有用，防止过拟合
- Lasso：特征选择，稀疏模型
- Elastic Net：两者结合

**评分要点：** 说出 L1 产生稀疏性得高分

## 题目 5：实战应用
**问题：** 如果发现线性回归模型欠拟合，你会采取哪些改进措施？

**参考答案：**
**改进措施：**
1. **增加特征**：添加相关特征
2. **多项式特征**：添加 x², x³等
3. **交互特征**：添加特征乘积
4. **减少正则化**：降低 alpha 值
5. **特征工程**：更好的特征表示
6. **换模型**：使用非线性模型（树、SVM 等）

**诊断方法：**
- 学习曲线：训练验证都低→欠拟合
- 残差图：有模式→非线性

**评分要点：** 说出 3 种以上方法，能说明诊断方法得高分
