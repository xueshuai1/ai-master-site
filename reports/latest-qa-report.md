## 控制台检查结果

| 页面 | 错误类型 | 详情 |
|------|----------|------|
| /blog | 404 | Failed to load resource: the server responded with a status of 404 |
| /roadmap | 404 | Failed to load resource: the server responded with a status of 404 |

分析：这是 Next.js RSC (React Server Components) 预取请求的 404，因为 blog 和 roadmap 页面都是 "use client" 纯客户端组件，没有对应的 RSC payload。功能不受影响，但会污染控制台。
