| **热度** | ⭐⭐⭐⭐⭐ 极高（AI 行业哲学分裂标志性事件） |

**深度分析：**

这是 2026 年 4 月最重要的单一事件——**同一天，两个截然相反的 AI 发布哲学碰撞**。

**Anthropic Claude Mythos（闭锁派）：**
- Anthropic 最强大的模型，能力"阶跃式提升"，在编码、学术推理和网络安全上远超 Claude Opus 4.6
- **不公开发布**，仅通过 **Project Glasswing** 向约 50 家关键基础设施合作伙伴提供受控访问
- 合作名单包括：AWS、Apple、Microsoft、Google、NVIDIA、Cisco、CrowdStrike、JPMorgan、Broadcom、Palo Alto Networks、Linux Foundation 等
- 用途：扫描自身系统和开源代码库中的可利用漏洞（包括隐藏了几十年的 bug）
- 定价：**$25/M input + $125/M output tokens**——史上最高
- 原因：Anthropic 明确表示该模型的网络攻击潜力对广泛开放太危险

**Zhipu AI GLM-5.1（开放派）：**
- 744B 参数 MoE 模型，每次前向传播仅 40B 活跃参数
- **MIT 许可证**——最宽松的开源许可，几乎无限制
- 200K 上下文窗口
- 在 **SWE-Bench Pro** 上据报同时击败 Claude Opus 4.6 和 GPT-5.4
- API 定价仅 **~$1/$3.2 per M tokens**，可完全免费自托管
- Zhipu 的路线：GLM-4.5 → 4.6 → 4.7 → 5 → 5.1，每一步都更开放、更强大

**核心叙事：** 这不是基准竞赛，不是价格战。这是行业最深层次的哲学分裂——最强大的模型被构建的速度，已经超过了任何人能就"谁该使用它"达成共识的速度。
