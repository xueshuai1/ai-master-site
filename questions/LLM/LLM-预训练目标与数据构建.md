## 题目描述
LLM 预训练的目标函数是什么？训练数据如何构建和处理？

**来源：** LLM 面试题整理
**标签：** 预训练，语言建模，数据清洗，词表

## 参考答案

**预训练目标：**
- 自回归语言建模：最大化 P(x_t | x_1, ..., x_{t-1})
- 损失函数：交叉熵损失 L = -Σ log P(x_t | context)
- 下一个 token 预测任务

**数据构建流程：**

1. **数据收集：**
   - 网页爬取（CommonCrawl 等）
   - 书籍、维基百科、代码仓库
   - 多语言、多领域覆盖

2. **数据清洗：**
   - 去重（文档级、句子级）
   - 过滤低质量内容
   - 移除 PII 敏感信息
   - 语言识别与分类

3. **分词处理：**
   - BPE/WordPiece/SentencePiece
   - 词表大小通常 32k-100k
   - 平衡词表覆盖率与序列长度

4. **打包与截断：**
   - 拼接成固定长度序列
   - 不跨越文档边界（可选）

## 考察重点
- **理解深度：** 预训练目标的数学表达
- **知识广度：** 数据处理的完整流程
- **应用能力：** 数据质量对模型的影响
