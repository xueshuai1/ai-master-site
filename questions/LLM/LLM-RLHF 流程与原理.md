## 题目描述
详细说明 RLHF（人类反馈强化学习）的完整流程及各阶段的作用。

**来源：** LLM 面试题整理
**标签：** RLHF，奖励模型，PPO，对齐

## 参考答案

**RLHF 三阶段：**

1. **监督微调（SFT）：**
   - 使用高质量<指令，回复>对微调
   - 让模型学会遵循指令的格式
   - 为后续阶段打下基础

2. **奖励模型训练（Reward Modeling）：**
   - 收集人类偏好数据（A/B 选择）
   - 训练 RM 预测人类偏好分数
   - 目标：RM 输出与人类判断一致

3. **强化学习优化（RL）：**
   - 使用 PPO 等算法优化策略
   - 奖励 = RM 分数 - KL 散度惩罚
   - KL 惩罚防止偏离 SFT 模型太远

**关键设计：**
- 奖励模型需要足够大以捕捉细微偏好
- KL 系数平衡对齐与多样性
- PPO 的 clip 机制保证训练稳定
