## 题目描述
设计一个 LLM Agent 系统，说明其核心组件和工作流程。

**标签：** Agent，工具使用，规划，记忆

## 参考答案

**Agent 核心组件：**

1. **规划（Planning）：**
   - 任务分解：复杂任务拆分为子任务
   - 反思：根据执行结果调整计划
   - ReAct：推理与行动交替

2. **工具使用（Tool Use）：**
   - API 调用：搜索、计算、代码执行
   - 函数调用：结构化参数传递
   - 工具选择：根据任务匹配工具

3. **记忆（Memory）：**
   - 短期记忆：当前对话上下文
   - 长期记忆：向量数据库存储
   - 工作记忆：任务执行状态

4. **行动执行：**
   - 工具调用与结果解析
   - 错误处理与重试
   - 多轮交互管理

**工作流程：**
1. 接收用户请求
2. 分析任务，制定计划
3. 选择工具，执行行动
4. 观察结果，更新记忆
5. 迭代直到任务完成
6. 生成最终回复

**应用框架：**
- LangChain、LlamaIndex
- AutoGen 多 Agent 协作
- OpenAI Function Calling

## 考察重点
- **理解深度：** Agent 各组件的设计原理
- **知识广度：** Agent 系统的完整架构
- **应用能力：** 实际 Agent 应用的开发
