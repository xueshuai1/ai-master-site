## 题目描述
对比全量微调、LoRA、QLoRA 等微调方法的原理、优缺点和适用场景。

**标签：** 微调，全量微调，LoRA，指令微调

## 参考答案

**全量微调（Full Fine-tuning）：**
- 更新所有参数
- 优点：性能最好，适应性强
- 缺点：显存需求大，易灾难性遗忘
- 适用：资源充足，任务差异大

**LoRA（Low-Rank Adaptation）：**
- 冻结原权重，注入低秩矩阵ΔW = BA
- 只训练 A 和 B（秩 r<<d）
- 优点：参数量少，多任务切换方便
- 缺点：可能欠拟合复杂任务
- 适用：资源受限，多任务场景

**QLoRA：**
- LoRA + 4bit 量化 + 分页优化器
- 优点：单卡可微调 65B 模型
- 缺点：训练速度略慢
- 适用：消费级显卡，大模型微调

**指令微调（Instruction Tuning）：**
- 使用<指令，输出>对进行 SFT
- 提升模型遵循指令的能力
- 是 RLHF 的前置步骤

## 考察重点
- **理解深度：** 各方法的数学原理
- **知识广度：** 方法对比与选型依据
- **应用能力：** 实际场景中的微调策略
