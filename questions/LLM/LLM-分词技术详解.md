## 题目描述
说明 BPE、WordPiece、SentencePiece 等分词算法的原理和区别。

**来源：** LLM 面试题整理
**标签：** 分词，BPE，WordPiece，词表

## 参考答案

**1. BPE（Byte Pair Encoding）：**
- 从字符级词表开始，反复合并最频繁的词对
- 合并次数决定词表大小
- GPT 系列使用
- 优点：平衡词表大小与覆盖率

**2. WordPiece：**
- BPE 的变体，选择最大化似然的合并
- 使用##前缀表示子词
- BERT 使用
- 与 BPE 效果相近，实现略有不同

**3. SentencePiece：**
- 直接处理原始文本，无需预分词
- 支持 BPE 和 Unigram 模式
- 将空格视为特殊字符
- 多语言场景友好

**关键考量：**
- 词表大小：32k-100k 常见
- 罕见词处理：UNK token 或分解为子词
- 多语言：需要覆盖多种字符集

## 考察重点

- **知识广度：** 主流分词方案对比
