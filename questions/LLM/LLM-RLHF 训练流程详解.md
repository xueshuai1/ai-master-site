## 题目描述
请详细解释 RLHF（Reinforcement Learning from Human Feedback）的训练流程，包括各阶段的目标和关键组件。

**来源：** LLM 面试题整理
**标签：** RLHF，人类反馈，PPO，对齐

## 参考答案

**RLHF 目标：**
将大模型与人类价值观对齐，使其输出更有帮助、诚实、无害。

**三阶段流程：**

**阶段 1：监督微调（SFT）**
```
预训练模型 + 高质量指令数据 → SFT 模型
```
- 收集人类编写的指令 - 回答对
- 监督微调，学习遵循指令

**阶段 2：奖励模型训练（Reward Modeling）**
```
SFT 模型生成多个回答 → 人类排序 → 训练奖励模型
```
- 对同一提示生成 4-9 个回答
- 人类标注偏好排序
- 训练 RM 预测人类偏好
- 损失函数：Ranking Loss

```python
# 奖励模型损失
loss = -log(σ(r(x, y_w) - r(x, y_l)))
# y_w: 偏好的回答，y_l: 不偏好的回答
```

**阶段 3：强化学习优化（PPO）**
```
SFT 模型 + 奖励模型 → PPO 训练 → 对齐模型
```
- Actor：SFT 模型（生成回答）
- Critic：价值函数（估计状态价值）
- Reward：奖励模型（给出分数）
- 使用 PPO 算法优化策略

**PPO 关键公式：**
```python
# 重要性采样比
r_t(θ) = π_θ(a_t|s_t) / π_θ_old(a_t|s_t)

# 裁剪目标
L^CLIP = E[min(r_t·A_t, clip(r_t, 1-ε, 1+ε)·A_t)]

# 最终损失
L = L^CLIP - c₁·价值损失 + c₂·熵奖励
```

**关键技巧：**
- KL 惩罚防止偏离 SFT 太远
- 优势函数归一化
- 梯度裁剪

**应用：**
- ChatGPT、Claude 等对齐训练
- 当前主流的大模型对齐方法
