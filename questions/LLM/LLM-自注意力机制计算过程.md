## 题目描述
详细说明自注意力机制的计算步骤，并分析其时间和空间复杂度。

**来源：** LLM 面试题整理
**标签：** 自注意力，QKV，计算复杂度，优化

## 参考答案

**计算步骤：**

1. **线性投影：**
   - 输入 X ∈ R^(n×d) 投影为 Q, K, V
   - Q = XW_Q, K = XW_K, V = XW_V

2. **注意力分数计算：**
   - 计算 Q 和 K 的点积：QK^T ∈ R^(n×n)
   - 缩放：除以√d_k

3. **Softmax 归一化：**
   - 每行做 softmax，得到注意力权重矩阵 A

4. **加权求和：**
   - 输出 = AV ∈ R^(n×d)

**复杂度分析：**
- 时间复杂度：O(n²d)，n 为序列长度，d 为维度
- 空间复杂度：O(n²) 存储注意力矩阵

**优化方案：**
- FlashAttention：减少 HBM 访问
- 稀疏注意力：降低 n²依赖
- 线性注意力：用核函数近似
