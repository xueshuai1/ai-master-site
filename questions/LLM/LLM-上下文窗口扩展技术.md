## 题目描述
列举并解释 LLM 扩展上下文窗口的主要技术方案。

**来源：** LLM 面试题整理
**标签：** 长上下文，窗口扩展，注意力优化，外推

## 参考答案

**1. 位置编码插值：**
- NTK-Aware 插值：调整 RoPE 频率
- 线性插值：缩放位置索引
- 支持 2-4 倍扩展，几乎无性能损失

**2. 注意力稀疏化：**
- 滑动窗口注意力：只关注局部窗口
- 全局 token + 局部注意力
- Mistral、Llama-3 使用

**3. 分层注意力：**
- 不同层使用不同注意力范围
- 浅层局部，深层全局

**4. 外部记忆：**
- RAG 检索相关片段
- 向量数据库存储长文档
- 突破模型原生窗口限制

**5. 架构改进：**
- Mamba/状态空间模型：线性复杂度
- RWKV：RNN 式 Transformer
- 理论上支持无限长度

## 考察重点
- **理解深度：** 各技术的原理与 trade-off
- **知识广度：** 长上下文的完整解决方案
- **应用能力：** 实际场景的技术选型
