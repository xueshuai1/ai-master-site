## 题目描述
说明 LLM 如何扩展为多模态模型，处理图像、视频等输入。

**来源：** LLM 面试题整理
**标签：** 多模态，视觉语言，图像编码，对齐

## 参考答案

**多模态架构设计：**

1. **视觉编码器：**
   - ViT/CLIP 编码图像为向量
   - 冻结或微调预训练权重
   - 输出视觉 token 序列

2. **投影层：**
   - 将视觉特征映射到语言空间
   - 线性投影或 Q-Former（BLIP-2）
   - 对齐视觉和语言表示

3. **融合方式：**
   - 早期融合：视觉 token 拼接到文本前
   - 交叉注意力：视觉作为 KV，文本作为 Q
   - 深层融合：多层交互

**代表模型：**
- LLaVA：线性投影 + Vicuna
- GPT-4V：闭源，推测类似架构
- Flamingo：Perceiver Resampler

**训练策略：**
- 两阶段：先对齐，再指令微调
- 数据：图像 - 文本对 + 多模态指令
