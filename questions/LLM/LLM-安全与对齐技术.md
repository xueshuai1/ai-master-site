## 题目描述
说明 LLM 面临的安全风险及对齐技术的实现方法。

**来源：** LLM 面试题整理
**标签：** 安全，对齐，红队，内容过滤

## 参考答案

**安全风险：**

1. **有害内容生成：**
   - 暴力、仇恨、歧视性言论
   - 危险活动指导
   - 虚假信息传播

2. **越狱攻击：**
   - Prompt 注入绕过安全限制
   - 角色扮演绕过
   - 多轮对话逐步诱导

3. **隐私泄露：**
   - 训练数据中的 PII 信息
   - 成员推断攻击

**对齐技术：**

1. **训练阶段：**
   - SFT 使用安全示范数据
   - RLHF 引入安全奖励信号
   - 对抗训练提升鲁棒性

2. **推理阶段：**
   - 输入过滤：检测恶意 prompt
   - 输出过滤：拦截有害内容
   - 安全分类器实时判断

3. **持续监控：**
   - 红队测试发现漏洞
   - 用户反馈收集
   - 定期安全更新

## 考察重点
- **理解深度：** 安全风险的分类与成因
- **知识广度：** 对齐技术的完整方案
- **应用能力：** 生产系统的安全设计
