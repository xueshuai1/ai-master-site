## 题目描述
什么是 LLM 幻觉？分析其成因并列举缓解方法。

**来源：** LLM 面试题整理
**标签：** 幻觉，事实性，RAG，校准

## 参考答案

**幻觉定义：**
- 模型生成看似合理但事实错误的内容
- 包括捏造事实、错误引用、矛盾陈述

**成因分析：**
1. 训练目标是最小化预测损失，而非保证真实性
2. 训练数据包含错误和矛盾信息
3. 模型缺乏事实核查能力
4. 解码策略倾向于流畅性而非准确性

**缓解方法：**

1. **检索增强生成（RAG）：**
   - 从外部知识库检索相关信息
   - 基于检索内容生成回答
   - 可追溯信息来源

2. **工具使用：**
   - 调用搜索引擎、计算器、代码解释器
   - 将事实性问题外包给可靠工具

3. **训练优化：**
   - 使用高质量、事实准确的数据
   - 引入事实性奖励信号（RLHF）
   - 训练模型表达不确定性

4. **解码策略：**
   - 降低温度，使用核采样
   - 多次采样一致性检查

## 考察重点
- **理解深度：** 幻觉的根本成因
- **知识广度：** 缓解方案的技术细节
- **应用能力：** 生产系统中的幻觉治理
