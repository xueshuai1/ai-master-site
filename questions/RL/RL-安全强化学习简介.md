## 题目描述
安全强化学习（Safe RL）旨在保证智能体行为的安全性，请解释安全 RL 的核心问题和主要方法。

**标签：** 安全 RL, 约束 RL, 风险控制，实际应用

## 参考答案

**核心问题：**
在最大化奖励的同时，满足安全约束，避免危险行为。

**形式化：**
- 标准 RL: max E[Σγ^t r_t]
- 安全 RL: max E[Σγ^t r_t], s.t. E[Σγ^t c_t] ≤ d
  - c_t: 成本函数（危险程度）
  - d: 成本上限

**主要方法：**

1. **约束策略优化（CPO）：**
   - 在策略更新时加入约束
   - 理论保证满足约束
   - 计算复杂

2. **拉格朗日方法：**
   - 将约束加入目标函数
   - L = 奖励 - λ·成本
   - 学习拉格朗日乘子λ

3. **安全层（Safety Layer）：**
   - 在策略输出后加安全过滤
   - 投影到安全动作空间
   - 简单有效

4. **风险敏感 RL：**
   - 优化风险度量（CVaR 等）
   - 而不仅是期望回报

5. **验证与监控：**
   - 运行时监控
   - 紧急停止机制

**应用场景：**
- 机器人：避免碰撞、自损伤
- 自动驾驶：遵守交通规则
- 医疗：避免伤害患者
- 金融：控制风险敞口

**挑战：**
- 约束定义困难
- 探索与安全的权衡
- 理论保证与实际效果的 gap

## 延伸追问
### 追问 1: 探索安全
**问题：** 如何在保证安全的前提下探索？
**答案：** 安全探索集、保守初始策略、模拟探索、人类监督等。

### 追问 2: 约束违反处理
**问题：** 训练过程中约束违反了怎么办？
**答案：** 惩罚、回滚、调整λ、限制更新幅度等。
