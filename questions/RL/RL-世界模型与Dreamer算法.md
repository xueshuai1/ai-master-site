## 题目描述
世界模型（World Models）和 Dreamer 是基于模型的 RL 的重要工作，请解释其核心思想和架构。

**来源：** 强化学习面试题整理
**标签：** 世界模型，Dreamer, Model-Based RL, 潜在空间

## 参考答案

**世界模型核心思想：**
学习环境的压缩表示（潜在空间），在潜在空间中进行"想象"和规划。

**Dreamer 架构：**

1. **RSSM（Recurrent State Space Model）：**
   - 学习潜在状态表示
   - 预测下一状态和奖励
   - 结合确定性 + 随机性状态

2. **编码器：**
   - 将观测编码为潜在状态
   - CNN + RNN

3. **解码器：**
   - 从潜在状态重建观测
   - 验证表示质量

4. **Critic 和 Actor：**
   - 在潜在空间中学习
   - 用想象轨迹训练

**训练流程：**
1. 收集真实经验
2. 训练世界模型（预测）
3. 在模型中"想象"轨迹
4. 用想象轨迹训练 Actor-Critic

**优势：**
- 样本效率极高
- 可以"做梦"学习
- 规划能力强

**Dreamer 版本：**
- DreamerV1: 基础版本
- DreamerV2: 改进 RSSM
- DreamerV3: 通用智能体

**与 Model-Free 对比：**
- 样本效率：Dreamer 高 10-100 倍
- 计算成本：Dreamer 更高
- 渐近性能：相当

## 延伸追问
### 追问 1: 潜在空间
**问题：** 为什么用潜在空间而不是原始状态？
**答案：** 压缩冗余信息、去除噪声、便于预测和规划。

### 追问 2: 想象轨迹
**问题：** "想象"轨迹是什么？
**答案：** 用学到的世界模型 rollout，无需与环境交互，生成合成经验。

## 深入理解
世界模型代表了 Model-Based RL 的最先进水平，是样本效率最高的方法之一。
