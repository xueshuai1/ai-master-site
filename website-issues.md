### 2. 知识库文章无法访问

**问题描述**: 点击知识库文章链接显示"文章不存在 Article not found"

**复现路径**: 
1. 访问 `/knowledge`
2. 点击任意推荐文章（如"Prompt Engineering 入门"）
3. 跳转到 `/knowledge/LLM/llm-001`
4. 显示"文章不存在"

**根本原因**: 
- 首页使用硬编码的示例 ID (`llm-001`, `ml-001`)
- 实际数据文件 ID 格式不一致 (`LLM-KB-001`, `LLM-001-RAG 基础概念与工作原理.json`)
- API 无法通过 `llm-001` 找到对应的 JSON 文件

**修复建议**:
- 方案 A: 重命名数据文件，使用统一的小写 ID 格式 (`llm-001.json`)
- 方案 B: 修改首页示例数据，使用真实的文章 ID
- 方案 C: 添加 ID 映射层，支持多种 ID 格式
