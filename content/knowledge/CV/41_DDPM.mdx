## 概述

DDPM（去噪扩散概率模型）是由 Ho 等人于 2020 年提出的扩散模型，通过简化的训练目标和高效的采样算法，实现了与 GAN 相媲美的生成质量，推动了扩散模型的广泛应用。

## 核心贡献

### 简化训练目标

**原始扩散模型：** 复杂的变分下界

**DDPM：** 简化为噪声预测

$$L_{simple} = \mathbb{E}_{t, x_0, \epsilon}[||\epsilon - \epsilon_\theta(x_t, t)||^2]$$

### 固定方差

$$\Sigma_\theta(x_t, t) = \sigma_t^2 I$$

其中 $\sigma_t^2 = \beta_t$ 或 $\tilde{\beta}_t = \frac{1 - \bar{\alpha}_{t-1}}{1 - \bar{\alpha}_t} \beta_t$

## 实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class DDPM(nn.Module):
    def __init__(self, unet, img_size=32, timesteps=1000):
        super().__init__()
        self.unet = unet
        self.img_size = img_size
        self.timesteps = timesteps
        
        # 噪声调度
        self.betas = torch.linspace(1e-4, 2e-2, timesteps)
        self.alphas = 1 - self.betas
        self.alphas_cumprod = torch.cumprod(self.alphas, dim=0)
        
        # 采样相关参数
        self.register_buffer('sqrt_alphas_cumprod', torch.sqrt(self.alphas_cumprod))
        self.register_buffer('sqrt_one_minus_alphas_cumprod', torch.sqrt(1 - self.alphas_cumprod))
        self.register_buffer('sqrt_recip_alphas_cumprod', torch.sqrt(1 / self.alphas_cumprod))
        self.register_buffer('sqrt_recipm1_alphas_cumprod', torch.sqrt(1 / self.alphas_cumprod - 1))
        
        # 采样方差
        self.register_buffer('posterior_variance', self.betas * (1 - self.alphas_cumprod[:-1]) / (1 - self.alphas_cumprod[1:]))
    
    def forward(self, x0):
        """训练：预测噪声"""
        batch_size = x0.shape[0]
        
        # 随机时间步
        t = torch.randint(0, self.timesteps, (batch_size,), dtype=torch.long)
        
        # 添加噪声
        noise = torch.randn_like(x0)
        sqrt_alpha = self.sqrt_alphas_cumprod[t].view(-1, 1, 1, 1).to(x0.device)
        sqrt_one_minus_alpha = self.sqrt_one_minus_alphas_cumprod[t].view(-1, 1, 1, 1).to(x0.device)
        
        xt = sqrt_alpha * x0 + sqrt_one_minus_alpha * noise
        
        # 预测噪声
        predicted_noise = self.unet(xt, t)
        
        # 损失
        loss = F.mse_loss(predicted_noise, noise)
        
        return loss
    
    @torch.no_grad()
    def sample(self, num_samples=1, device='cuda'):
        """采样：从噪声生成图像"""
        # 纯噪声
        x = torch.randn(num_samples, 3, self.img_size, self.img_size, device=device)
        
        for t in reversed(range(self.timesteps)):
            t_batch = torch.full((num_samples,), t, dtype=torch.long, device=device)
            
            # 预测噪声
            predicted_noise = self.unet(x, t_batch)
            
            # 计算 x0 预测
            alpha = self.alphas_cumprod[t].to(device)
            x0_pred = (x - torch.sqrt(1 - alpha) * predicted_noise) / torch.sqrt(alpha)
            x0_pred = x0_pred.clamp(-1, 1)
            
            # 计算均值
            if t > 0:
                alpha_prev = self.alphas_cumprod[t-1].to(device)
                beta = self.betas[t].to(device)
                
                mean = torch.sqrt(alpha_prev) * (1 - beta) / (1 - alpha) * x + \
                       torch.sqrt(beta) * (1 - alpha_prev) / (1 - alpha) * x0_pred
                
                # 添加噪声
                variance = torch.sqrt(beta) * torch.randn_like(x)
                x = mean + variance
            else:
                x = x0_pred
        
        return x
```

```python
    weights = losses / losses.sum()
    return torch.multinomial(weights, batch_size, replacement=True)
```
