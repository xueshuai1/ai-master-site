## 参考答案

### SVM 核心思想

**目标：** 找到一个超平面，最大化不同类别之间的间隔 (Margin)。

**关键概念：**
- **支持向量：** 距离超平面最近的样本点
- **间隔 (Margin)：** 支持向量到超平面的距离
- **硬间隔：** 所有样本都正确分类（线性可分）
- **软间隔：** 允许部分样本分类错误（线性不可分）

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn import svm
from sklearn.datasets import make_blobs

# 生成数据
X, y = make_blobs(n_samples=50, centers=2, cluster_std=0.8, random_state=42)

# 训练 SVM
clf = svm.SVC(kernel='linear', C=1.0)
clf.fit(X, y)

# 获取支持向量
support_vectors = clf.support_vectors_

# 可视化
plt.figure(figsize=(10, 8))

# 决策边界
xx, yy = np.meshgrid(np.linspace(X[:, 0].min() - 1, X[:, 0].max() + 1, 500),
                     np.linspace(X[:, 1].min() - 1, X[:, 1].max() + 1, 500))
Z = clf.decision_function(np.c_[xx.ravel(), yy.ravel()])
Z = Z.reshape(xx.shape)

plt.contour(xx, yy, Z, levels=[-1, 0, 1], colors=['blue', 'black', 'red'], 
            linestyles=['--', '-', '--'], linewidths=[2, 3, 2])
plt.contourf(xx, yy, Z, levels=[-1, 0, 1], colors=['blue', 'red'], alpha=0.1)

# 数据点
plt.scatter(X[:, 0], X[:, 1], c=y, cmap='viridis', s=50, edgecolors='k', alpha=0.6)

# 支持向量
plt.scatter(support_vectors[:, 0], support_vectors[:, 1], s=200, 
            facecolors='none', edgecolors='yellow', linewidths=3, 
            marker='o', label='Support Vectors')

plt.xlabel('Feature 1')
plt.ylabel('Feature 2')
plt.title('SVM with Support Vectors')
plt.legend()
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

print(f"Number of support vectors: {len(support_vectors)}")
print(f"Support vector indices: {clf.support_}")
```

### 核函数 (Kernel Functions)

```python
from sklearn.svm import SVC
from sklearn.preprocessing import StandardScaler

# 常用核函数对比
kernels = ['linear', 'poly', 'rbf', 'sigmoid']

fig, axes = plt.subplots(2, 2, figsize=(14, 12))
axes = axes.flatten()

# 生成非线性数据
from sklearn.datasets import make_circles
X_circles, y_circles = make_circles(n_samples=300, noise=0.1, factor=0.3, random_state=42)
X_circles = StandardScaler().fit_transform(X_circles)

for ax, kernel in zip(axes, kernels):
    if kernel == 'poly':
        clf = svm.SVC(kernel=kernel, degree=3, gamma='scale', C=1.0)
    else:
        clf = svm.SVC(kernel=kernel, gamma='scale', C=1.0)
    
    clf.fit(X_circles, y_circles)
    
    # 决策边界
    xx, yy = np.meshgrid(np.linspace(X_circles[:, 0].min() - 0.5, X_circles[:, 0].max() + 0.5, 200),
                         np.linspace(X_circles[:, 1].min() - 0.5, X_circles[:, 1].max() + 0.5, 200))
    Z = clf.decision_function(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)
    
    ax.contourf(xx, yy, Z, levels=50, cmap='RdBu', alpha=0.3)
    ax.scatter(X_circles[:, 0], X_circles[:, 1], c=y_circles, cmap='viridis', 
               s=30, edgecolors='k', alpha=0.6)
    ax.set_title(f'Kernel: {kernel}')
    ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

# 核函数公式
kernel_formulas = """
核函数对比：

1. Linear (线性核):
   K(x, x') = x · x'
   适用：线性可分数据

2. Polynomial (多项式核):
   K(x, x') = (γ × x · x' + r)^d
   参数：degree=d, gamma=γ, coef0=r
   适用：特征交互重要

3. RBF (径向基核，高斯核):
   K(x, x') = exp(-γ × ||x - x'||²)
   参数：gamma=γ
   适用：大多数非线性问题（默认推荐）

4. Sigmoid (S 型核):
   K(x, x') = tanh(γ × x · x' + r)
   适用：神经网络替代
"""
print(kernel_formulas)
```

### 参数调优

```python
from sklearn.model_selection import GridSearchCV

# 网格搜索
param_grid = {
    'C': [0.1, 1, 10, 100],
    'gamma': ['scale', 0.001, 0.01, 0.1, 1],
    'kernel': ['rbf']
}

grid_search = GridSearchCV(
    SVC(),
    param_grid,
    cv=5,
    scoring='accuracy',
    n_jobs=-1
)

grid_search.fit(X_circles, y_circles)

print(f"Best parameters: {grid_search.best_params_}")
print(f"Best CV score: {grid_search.best_score_:.4f}")

# C 和 gamma 的影响
# C: 正则化参数
# - C 大：低偏差，高方差（硬间隔）
# - C 小：高偏差，低方差（软间隔）

# gamma: 核系数
# - gamma 大：低偏差，高方差（复杂边界）
# - gamma 小：高偏差，低方差（简单边界）
```

### SVM 优缺点

| 优点 | 缺点 |
|------|------|
| 在高维空间有效 | 大数据集训练慢 |
| 内存高效（只用支持向量） | 多分类需要特殊处理 |
| 核技巧处理非线性 | 对参数和核选择敏感 |
| 泛化能力强 | 概率输出需要额外计算 |
| 理论基础强 | 缺失值处理困难 |

## 延伸追问

### 追问 1: 为什么 SVM 只用支持向量？

**问题：** 支持向量的特殊意义是什么？

**答案：**

```python
# 支持向量是决定超平面的关键样本
# 其他样本不影响最终模型

# 数学解释：
# SVM 的对偶问题：
# max Σα_i - ½ΣΣα_iα_j y_i y_j K(x_i, x_j)
# s.t. Σα_i y_i = 0, α_i ≥ 0

# 只有支持向量的 α_i > 0
# 非支持向量的 α_i = 0

# 决策函数：
# f(x) = Σ(α_i y_i K(x_i, x)) + b
#      = Σ_sv(α_i y_i K(x_i, x)) + b  (只含支持向量)

# 实际意义：
# 1. 模型稀疏（存储高效）
# 2. 预测快速（只计算支持向量）
# 3. 鲁棒性强（移除非 SV 不影响模型）
```

### 追问 2: 如何处理多分类问题？

**问题：** SVM 原生是二分类，如何扩展到多分类？

**答案：**

```python
from sklearn.svm import SVC

# 方法 1: One-vs-Rest (OvR)
# 为每个类别训练一个 SVM（该类 vs 其他）
clf_ovr = SVC(kernel='rbf', decision_function_shape='ovr')
clf_ovr.fit(X, y_multi)

# 方法 2: One-vs-One (OvO)
# 为每对类别训练一个 SVM
clf_ovo = SVC(kernel='rbf', decision_function_shape='ovo')
clf_ovo.fit(X, y_multi)

# sklearn 默认使用 OvO
# OvR 适合类别少的情况
# OvO 适合样本少的情况

# 方法 3: 直接使用多分类 SVM
# Crammer-Singer 方法
from sklearn.multiclass import OutputCodeClassifier

clf_code = OutputCodeClassifier(SVC(kernel='rbf'))
clf_code.fit(X, y_multi)
```

## 深入理解

**SVM 的核心：** 间隔最大化 + 核技巧

**核技巧的本质：** 隐式映射到高维，避免显式计算

**适用场景：** 中小数据集、高维特征、需要强泛化
